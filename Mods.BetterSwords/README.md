# Better Swords

**Author:** Nakai  
**Version:** 1.0.0  
**Game:** Hytale Early Access

## Description

Better Swords enhances sword combat with visual effects and improved attacks!

## Features

### Sword Trail Effects
- Every sword swing now displays a beautiful charged trail effect
- Trails follow your blade during the swing animation

### Enhanced Impact Particles
- Impact_Sword_Charged - Stylish charged sword impact effect
- Impact_Blade_01 - Additional blade slash effect
- More visible and satisfying hit feedback!

### Omnislash Signature
- Replaces the default VortexStrike signature
- Multi-hit combo attack with 4 powerful strikes
- Each strike deals 18 Physical damage
- Lock-on to target during the attack

### Improved Thrust Attack
- Faster charge time (0.3s instead of 0.7s)
- More powerful dash (Force: 100)
- Quick stop with high resistance

## Installation

1. Download the ZIP file
2. Extract to %APPDATA%\Hytale\UserData\Mods\
3. Make sure the folder is named Nakai.BetterSwords
4. Launch Hytale and enjoy!

## Compatibility

- Hytale Early Access (January 2026)
- Compatible with other mods that dont modify sword interactions
